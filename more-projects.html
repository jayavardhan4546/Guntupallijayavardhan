Projects

Chemalyze

Chemalyze is a full-stack web application that predicts the primary uses, - benefits, and disadvantages of chemicals from uploaded images. It features a React.js frontend and a Node.js backend, integrating OCR (Tesseract & PyTesseract) to extract chemical names, which are then analyzed using a Random Forest machine learning model. The app is containerized with Docker for easy deployment and includes all components—OCR setup, model inference, and result display—within a seamless workflow. Users simply upload an image, and the app returns insightful predictions about the detected chemical.





CallPrep Ai Agent

I developed an AI agent that automates pre-call research for sales reps by generating structured reports from public data. It scrapes company websites, analyzes LinkedIn profiles, and compiles insights using LLaMA 3 and GPT-4o mini. Built with Relevance AI, this tool saves time and boosts meeting preparedness.





TACLENS (Terms and conditions summarizer)

I developed a browser extension that simplifies complex legal documents—like user agreements and privacy policies—into clear, concise summaries. Powered by AI, it extracts key points from lengthy terms and conditions, helping users understand crucial information in seconds without reading the entire document.





Security for SCADA Systems

This project simulates a SCADA (Supervisory Control and Data Acquisition) environment to study and address security vulnerabilities. It includes the implementation of attack scenarios like man-in-the-middle (MITM) and introduces defense mechanisms such as checksum validation to detect and mitigate data tampering within industrial communication networks.





Sentiment Analysis

A Flask-based NLP tool that performs real-time sentiment analysis using NLTK’s VADER. It preprocesses user-input text and outputs both sentiment classification and percentage scores across positive, negative, and neutral categories. The app features a clean web interface, combining efficient text processing with intuitive visual feedback.





Logo Detection

This project focuses on detecting brand logos in images using the YOLOv4 (You Only Look Once) object detection algorithm. It leverages deep learning to accurately locate and classify multiple logos in real-time, making it suitable for applications like brand monitoring, advertising analytics, and visual content filtering.
